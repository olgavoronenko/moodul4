# Moodul 4 — Nurr WordPress CMS

See projekt muudab III mooduli staatilise Nurr kassikohviku lehe WordPressi CMS-i teemaks ja lisab eraldi plugina kasside haldamiseks.

## Struktuur

- `wp-content/themes/nurr-theme` — custom WordPress theme
- `wp-content/plugins/nurr-cats` — custom plugin kasside lisamiseks
- `docs` — dokumentatsioon ja juhendid
